What is Zsiistant?

Zsiistant is a factory-made OpenClaw/PicoClaw clone that gives you plug-and-play AI agent management. Built for power users who want full control and non-tech users who want guided pathways. One package, two modes: Unfiltered for raw power, Preset Mode for guided use cases.

Zero Dependencies Node.js Only SQLite Built-in Vanilla Web UI Local-First 50 Providers
install.bat

Quick Start

# Clone & run in 30 seconds
$ git clone https://github.com/zarigata/AC.git zsiistant
$ cd zsiistant
$ npm install
$ npm start

# Open in browser
$ open http://localhost:4000
features.dll

Core Features

🤖 100 Agents

Run up to 100 agents per machine, each isolated by default with opt-in communication.

🔌 50 Providers

Ollama, OpenAI, Anthropic, Z.AI, and 46 more. Fallback chains and auto-failover.

💰 Token Discipline

Better outcomes per token. Budget controls, smart routing, caching, prompt optimization.

📡 Multi-Channel

Telegram, Discord, Signal, WhatsApp, Email, Web — all from one agent.

🧠 Agent Memory

Short-term, long-term, episodic, and semantic memory with RAG built-in.

⚡ Zero Deps

Node.js + SQLite. No PostgreSQL, no Redis, no Docker. One command to run.
