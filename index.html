Install in seconds

🚀 One-liner

curl -sSL https://getzazi.dev | bash

Detects Docker, falls back to bare-metal, writes a systemd service. You’re done.

🐳 Docker Compose

git clone https://github.com/zarigata/AC.git zazi
cd zazi
docker-compose up -d

Port 4000. Named volume zazi-data:/data. Restart unless-stopped.

📦 Bare Metal

git clone https://github.com/zarigata/AC.git zazi
cd zazi
npm install
npm start

Node 22+ required. No other dependencies. Open http://localhost:4000.
